# Design System — Финансовый автопилот

Продукт: SaaS AI CFO для МСБ в РФ.
Характер: **финансовый, доверительный, ясный** — не startup-playful, не BI-тяжёлый.
Пользователь тратит 3 минуты утром. Всё критичное — без скролла.

---

## Иерархия главного экрана

Приоритет элементов (порядок считывания взглядом):

```
┌──────────────────────────────────────────┐
│  [Логотип / Компания]              [08:00]│  ← header, нейтральный
├──────────────────────────────────────────┤
│                                          │
│   ₽ 4 820 000          ▲ +3% за 7 дней  │  ← 1. ОСТАТОК hero (text-3xl bold)
│                                          │
│   ⚠ Кассовый разрыв через 11 дней       │  ← 2. ДЕФИЦИТ badge (alert) если < 14д
│                                          │
├──────────────────────────────────────────┤
│  БЛИЖАЙШИЕ ОБЯЗАТЕЛЬСТВА                 │  ← 3. NEXT 3 obligations
│  → ООО Ромашка   15 000 ₽  через 3 дня  │
│  → Аренда        80 000 ₽  через 5 дней │
│  → Зарплата     450 000 ₽  через 8 дней │
├──────────────────────────────────────────┤
│  [Sparkline прогноза 30 дней]            │  ← 4. только если вмещается
└──────────────────────────────────────────┘
```

Правило: все 4 блока должны помещаться на экране 375×667px (iPhone SE) без скролла.

## Interaction States

| Экран / Feature | Empty | Loading | Error |
|---|---|---|---|
| Главный экран | «Загрузите банковскую выписку» + CTA «Загрузить» (trust-кнопка) | Skeleton: 3 серые строки без цифр | «Не удалось загрузить данные» + кнопка Retry |
| Прогноз | «Добавьте обязательства — прогноз станет точнее» | Spinner по центру блока | «Не удалось рассчитать прогноз» |
| Список транзакций | «Нет транзакций за выбранный период» | Skeleton-список (5 строк) | «Ошибка загрузки» + Retry |
| AI-чат | «Задайте вопрос: "Почему упал остаток на прошлой неделе?"» | Typing indicator (3 точки) | «Не удалось получить ответ. Попробуйте позже» |
| Загрузка файла | — | Прогресс-бар + «Обрабатываем…» | Текст ошибки построчно из `import_batch.error_log` |

## Стейты Stale Data

Если `now - last_import_at > 24 часов`:
- Badge «Данные устарели» (warn-цвет) рядом с остатком
- Текст: «Обновлено: 2 дня назад» (не технический timestamp)

---

## Компоненты — анти-паттерны (запрещено)

- Стопка карточек как основной layout (stacked cards)
- Generic 3-колоночная feature grid в главном экране
- Chart без единицы измерения и подписи оси
- Цвет без семантического смысла (синий = просто синий)
- Spinner без текста при загрузке > 1 сек
