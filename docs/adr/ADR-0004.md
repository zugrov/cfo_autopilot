# ADR-0004: LLM Strategy — OpenRouter (Claude Sonnet) primary, YandexGPT fallback

**Статус:** Принято  
**Дата:** 2026-05-23  
**Авторы:** Engineering Review

---

## Контекст

Продукт включает AI-чат (Sprint 3): ~20 типовых вопросов о финансовом состоянии компании.
Требования: ответ ≤15 с (ТЗ §9.3), данные клиента не утекают, работает в РФ-контуре при необходимости.

Варианты:
1. **OpenAI GPT-4o напрямую** — качество высокое, нет гарантий доступности из РФ.
2. **YandexGPT** — российский провайдер, данные в РФ, но качество для финансового reasoning уступает.
3. **OpenRouter (Claude Sonnet) + YandexGPT fallback** — лучшее качество через роутинг, fallback для надёжности.

---

## Решение: OpenRouter (Claude Sonnet) + YandexGPT fallback

```
LLMAdapter
  ├─ primary:  OpenRouter → claude-sonnet-4-5
  └─ fallback: YandexGPT (при деградации OpenRouter)
```

Выбор per company в `company.settings_json`:
```json
{"llm_provider": "openrouter"}  // default
{"llm_provider": "yandexgpt"}   // для клиентов с требованием РФ-контура
```

Автофallback: если OpenRouter p95 latency > 12 с или error rate > 5% — переключение на YandexGPT на 10 минут.

---

## PII и безопасность

- Поле `purpose` банковской транзакции может содержать ФИО, ИНН, паспортные данные.
- Перед отправкой в LLM: `redact_pii(text)` — маскирование по regex (ИНН 10/12 цифр, паспорт XX XX XXXXXX).
- RAG retrieval фильтрует строго по `company_id` — cross-tenant retrieval запрещён.
- Кеш ответов: ключ `(question_hash, data_version)`, где `data_version = max(import_batch.updated_at)`.

---

## Обоснование

- Claude Sonnet: лучший reasoning для структурированных финансовых вопросов (SQL generation, explain).
- OpenRouter: абстракция над провайдерами, не зависит от хостинга.
- YandexGPT fallback: для клиентов с compliance-требованием «данные только в РФ».
- Не GPT-4o directly: нестабильная доступность из РФ без VPN.

---

## Последствия

- OpenRouter API key в env (`.env`), не в коде.
- YandexGPT endpoint: `https://llm.api.cloud.yandex.net/foundationModels/v1/completion`.
- Мониторинг p95 latency OpenRouter — метрика в production dashboard.
- YAML-каталог вопросов (`app/llm/questions.yaml`) — расширяемый, не хардкод.
